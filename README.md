# Festival DevOps Music Fest 2026

## Descripción
Landing page profesional para la promoción del festival, desplegada mediante una arquitectura de microservicios multi-contenedor.

## Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, Nginx.
* **Backend:** Python, Flask, MySQL Connector.
* **Base de Datos:** MySQL 8.0.
* **Infraestructura:** Docker, Docker Compose, Git y GitHub.

## Orquestación
Se utiliza `docker-compose.yml` para levantar la infraestructura de forma declarativa, gestionando la red privada y los volúmenes de persistencia de datos.
